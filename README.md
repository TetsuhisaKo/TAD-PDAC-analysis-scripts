# Analysis scripts for TAD-PDAC study

This repository contains analysis scripts accompanying the manuscript:

"Clinically defined tumor-associated diabetes identifies a T-cell-hypoinfiltrated phenotype in resected pancreatic ductal adenocarcinoma."

The scripts document the statistical and bioinformatic workflows used for survival analyses, differential expression analysis, gene set enrichment analysis, TCGA-PAAD analyses, ELISA quantification, and correlation analyses.

Patient-level clinical data are not included because of patient privacy and ethical restrictions. Public TCGA-PAAD data were obtained from the UCSC Xena GDC Hub. RNA-sequencing and whole-genome sequencing data generated in this study have been deposited in the European Genome-phenome Archive under accession number EGAS00001007212.

Local file names and column labels should be adjusted according to the accompanying data dictionary or local data structure.
